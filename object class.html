Object Class in C++

In C++, a class is a blueprint for creating objects that encapsulate data and behavior. An object is an instance of a class, and it can have its own data members and member functions.

Example:

Here's an example of defining a class and creating objects in C++: 

				
					#include <iostream>

					// Define a class named 'Person'
					class Person {
						private:
							std::string name;
							int age;

						public:
							void setName(std::string n) {
								name = n;
							}

							void setAge(int a) {
								age = a;
							}

							void displayInfo() {
								std::cout << "Name: " << name << std::endl;
								std::cout << "Age: " << age << std::endl;
							}
					};

					int main() {
						// Create objects of the 'Person' class
						Person person1;
						Person person2;

						// Set data for person1
						person1.setName("John Doe");
						person1.setAge(25);

						// Set data for person2
						person2.setName("Jane Smith");
						person2.setAge(30);

						// Display information of both persons
						std::cout << "Person 1:" << std::endl;
						person1.displayInfo();
						std::cout << std::endl;

						std::cout << "Person 2:" << std::endl;
						person2.displayInfo();

					return 0;
				}

The above code defines a class named 'Person' with private data members 'name' and 'age', and public member functions to set and display the information. In the 'main' function, two objects of the 'Person' class are created, and their data is set using the member functions. The information of both persons is then displayed using the 'displayInfo' function.

The output of the above code will be: 

			
				Person 1:
				Name: John Doe
				Age: 25

				Person 2:
				Name: Jane Smith
				Age: 30

With classes and objects, you can create reusable and modular code in C++. Objects allow you to represent real-world entities and perform operations on them using member functions.

For more details and advanced concepts of object-oriented programming in C++, you can refer to the C++ Object-Oriented Programming Documentation.
