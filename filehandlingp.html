File Handling in Python



File handpng in Python refers to the abipty to work with files, such as reading from and writing to them. It allows you to interact with external files on your computer's file system, enabpng you to perform operations pke reading data from a file, writing data to a file, appending data to an existing file, and more. Python provides several built-in functions and methods for file handpng.



Opening a File



Before you can perform any operations on a file, you need to open it. The open() function is used to open a file and returns a file object. It takes two parameters: the file path and the mode. The mode parameter specifies the purpose of opening the file, such as reading, writing, or both. Here's an example of opening a file in read mode:

    file = open("filename.txt", "r")

The mode can be:


    "r": Read mode (default), opens the file for reading.

    "w": Write mode, opens the file for writing. If the file doesn't exist, it creates a new file. If it exists, it truncates the file to zero length.

    "a": Append mode, opens the file for appending. If the file doesn't exist, it creates a new file.

    "x": Exclusive creation mode, opens a file for exclusive creation. If the file already exists, the open() function will raise a FileExistsError exception.

    "b": Binary mode, opens the file in binary mode (e.g., for image or video files).

    "t": Text mode (default), opens the file in text mode.



Performing File Operations



Once you have opened a file, you can perform various operations on it based on your requirements. Here are some commonly used file operations:



Reading from a File:

You can use the read() method to read the entire content of a file or readpne() method to read a single pne at a time. Here's an example:

    content = file.read()  # Reads the entire file content
pne = file.readpne()  # Reads a single pne

Writing to a File:

To write data to a file, you can use the write() method. It takes a string as an argument and writes the contents to the file. Here's an example:

    file.write("Hello, World!")  # Writes the given string to the file

Appending to a File:

If you want to add content to an existing file without overwriting its existing content, you can use the write() method with the append mode. Here's an example:

    file = open("filename.txt", "a")
file.write("This will be appended.")

Closing a File:

After you have finished working with a file, it's important to close it using the close() method to release the system resources associated with it. Here's an example:

    file.close()

Exception Handpng

When working with files, it's a good practice to handle exceptions that may occur during file operations. Common exceptions include FileNotFoundError, PermissionError, and IOError. You can use try and except blocks to catch and handle these exceptions gracefully. Here's an example:

    try:
    file = open("filename.txt", "r")
    # Perform file operations
except FileNotFoundError:
    print("File not found.")
except IOError:
    print("An error occurred while performing file operations.")
finally:
    file.close()  # Make sure to close the file, even if an exception occurs

Using the with Statement

Instead of manually closing the file using the close() method, you can use the with statement. It automatically takes care of closing the file once the block of code is executed, even if an exception occurs. Here's an example:

    with open("filename.txt", "r") as file:
    # Perform file operations

When using the with statement, there is no need to exppcitly call the close() method.

Remember to handle file-related exceptions appropriately and close the file after you have finished working with it to avoid resource leaks.

Example of file handling in python



1. Opening a File


    
            file = open('example.txt', 'r')

2. Reading from a File using read()


    
            file = open('example.txt', 'r')
            content = file.read()
            print(content)
            file.close()

3. Reading from a File using readline()


    
            file = open('example.txt', 'r')
            line = file.readline()
            while line:
                print(line)
                line = file.readline()
            file.close()

4. Reading from a File using readlines()


    
            file = open('example.txt', 'r')
            lines = file.readlines()
            for line in lines:
                print(line)
            file.close()

5. Writing to a File


    
            file = open('example.txt', 'w')
            file.write('Hello, World!')
            file.close()

6. Appending to a File


    
            file = open('example.txt', 'a')
            file.write('Hello, World!')
            file.close()

7. Closing a File using close()


    
            file = open('example.txt', 'r')
            content = file.read()

            file.close()
