Introduction to C++ -

C++ was developed in the 1980s as an extension of the C programming language. It is an object-oriented programming language, which means that it allows you to create objects that can contain data and methods. C++ is a compiled language, which means that the source code is compiled into machine code before it can be executed.



Getting Started -

To get started with C++, you will need a text editor and a C++ compiler. You can use any text editor that you are comfortable with, but we recommend using an editor that has syntax highlighting and auto-completion features, such as Visual Studio Code or Sublime Text. For the compiler, we recommend using a free and open-source compiler such as GCC or Clang.



Let's start with a simple "Hello, World!" example in C++. Open your text editor and create a new file called "hello.cpp". In this file, type the following code:


Hello World Example -

#include <iostream>

        int main() {
	        std::cout << "Hello, World!" << std::endl;
	        return 0;
            }

Save the file and open a command prompt or terminal window. Navigate to the directory where you saved the file and type the following command to compile the code:

  • g++ hello.cpp -o hello

This will compile the code and create an executable file called "hello". To run the program, type the following command:

  • ./hello

This will run the program and you should see the message "Hello, World!" printed to the screen.


Data Types -

C++ has several built-in data types, including:

  • int - for integers
  • float - for floating-point numbers
  • double - for double-precision floating-point numbers
  • char - for characters
  • bool - for Boolean values

You can declare variables of these data types using the following syntax:

  • int myInt = 42;
  • float myFloat = 3.14;
  • double myDouble = 3.14159;
  • char myChar = 'A';
  • bool myBool = true;

You can also declare arrays of these data types using the following syntax:


			int myIntArray[5] = {1, 2, 3, 4, 5};
			float myFloatArray[3] = {1.0, 2.0, 3.0};
			char myCharArray[6] = {'H', 'e', 'l', 'l', 'o', '0'};
