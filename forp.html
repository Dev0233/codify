For Loop in Python





A for loop is a control flow statement in Python that allows you to iterate over a sequence of values, such as a list, tuple, string, or dictionary. The general syntax of a for loop in Python is as follows:

    for variable in sequence:
        # do something

Here, variable is a variable that will take on each value in the sequence during each iteration of the loop. The indented block of code underneath the for statement is the body of the loop, which is executed once for each value in the sequence.


For example, let's say we have a list of numbers that we want to iterate over using a for loop:

    numbers = [1, 2, 3, 4, 5]
    for number in numbers:
        print(number)

In this example, we define a list of numbers [1, 2, 3, 4, 5], and then use a for loop to iterate over the list. During each iteration of the loop, the number variable takes on the value of the current element in the list, and the print function is called to display the value on the screen.


The range() function is commonly used to create a sequence of numbers to iterate over in a for loop:

    for i in range(1, 6):
        print(i)

In this example, we use the range() function to create a sequence of numbers from 1 to 5 (inclusive), and then use a for loop to iterate over the sequence. The i variable takes on the value of each number in the sequence during each iteration of the loop, and the print function is called to display the value on the screen.


In addition to iterating over sequences, you can also use a for loop to iterate over the keys and values in a dictionary:

    my_dict = {'a': 1, 'b': 2, 'c': 3}
    for key, value in my_dict.items():
        print(key, value)

In this example, we define a dictionary {'a': 1, 'b': 2, 'c': 3}, and then use a for loop to iterate over the items() in the dictionary. During each iteration of the loop, the key variable takes on the key of the current item in the dictionary, and the value variable takes on the value of the current item. The print function is called to display the output of the current key-value pair on the screen.


The for loop is used to iterate over a sequence of values in Python.



Basic Syntax




The basic syntax of the for loop in Python is:


    for variable in sequence:
        # Code to execute for each value in the sequence

For example:


    fruits = ["apple", "banana", "cherry"]
    for fruit in fruits:
        print(fruit)

This code will iterate over the values in the fruits list and print each one to the screen.



Advanced Features


There are several advanced features of the for loop in Python:

  • You can use the break statement to exit the loop early if a certain condition is met.
  • You can use the continue statement to skip the current iteration of the loop and move on to the next one.
  • You can use the else clause to execute some code after the loop has finished iterating over the sequence.
  • You can use the enumerate() function to iterate over both the values and the indices of a sequence.
  • You can use the zip() function to iterate over multiple sequences in parallel.

For example:


    for i in range(1, 6):
        if i == 3:
            break
        print(i)

This code will iterate over the numbers 1 to 5, but when i is equal to 3, the break statement will be executed and the loop will exit early.


    for i in range(1, 6):
        if i == 3:
            continue
        print(i)

This code will iterate over the numbers 1 to 5, but when i is equal to 3, the continue statement will be executed and the loop will skip over that iteration.


    fruits = ["apple", "banana", "cherry"]
    for i, fruit in enumerate(fruits):
        print(i, fruit)

This code uses the enumerate() function to iterate over the fruits list and print both the index and the value of each element to the screen.


    names = ["Alice", "Bob", "Charlie"]
    ages = [25, 30, 35]
    for name, age in zip(names, ages):
        print(name, age)

This code uses the zip() function to iterate over both the names and ages lists in parallel and print out each person's name and age.
