Arrays in Python



In Python, an array is a collection of elements of the same data type. Unlike lists, arrays are fixed in size and cannot be resized once created. Python doesn't have built-in support for arrays, but we can use the array module to create and manipulate arrays.

To use the array module, we first need to import it:

import array

We can then create an array by specifying the data type and initializing the values:

my_array = array.array('i', [1, 2, 3, 4, 5])

In this example, we have created an array of integers ('i') with the values 1, 2, 3, 4, and 5.

Here are some common data types that can be used with arrays:

  • 'b': signed integer (1 byte)
  • 'B': unsigned integer (1 byte)
  • 'h': signed integer (2 bytes)
  • 'H': unsigned integer (2 bytes)
  • 'i': signed integer (4 bytes)
  • 'I': unsigned integer (4 bytes)
  • 'f': floating point (4 bytes)
  • 'd': floating point (8 bytes)

Arrays in Python have several advantages over lists. Since arrays are of a fixed size, they are more memory-efficient than lists. Also, since all elements in an array are of the same data type, they can be processed more efficiently by the computer's CPU.

Examples

Here are some examples of common operations on arrays in Python:

    # Accessing elements in an array
	my_array = array.array('i', [1, 2, 3, 4, 5])
	print(my_array[0]) # Output: 1
	print(my_array[2]) # Output: 3
	Inserting elements into an array
	my_array.insert(2, 6)
	print(my_array) # Output: array('i', [1, 2, 6, 3, 4, 5])

	Removing elements from an array
	my_array.remove(3)
	print(my_array) # Output: array('i', [1, 2, 6, 4, 5])

	Slicing an array
	sliced_array = my_array[1:4]
	print(sliced_array) # Output: array('i', [2, 6, 4])

	Sorting an array
	my_array = array.array('i', [4, 1, 3, 2])
	my_array.sort()
	print(my_array) # Output: array('i', [1, 2, 3, 4])

In conclusion, arrays in Python are fixed-size collections of elements of the same data type. They can be more memory-efficient than lists and can be processed more efficiently by the CPU. The array module provides several functions for creating and manipulating arrays in Python.


Creating an array:


    import array
a = array.array('i', [1, 2, 3, 4, 5])

Adding an element to the end of the array:


    a.append(6)

Adding the elements of another array to the end of the array:


    b = array.array('i', [7, 8, 9])
	a.extend(b)

Inserting an element at a specific index:


    a.insert(0, 0)

Removing the last element of the array:


    a.pop()

Removing the first occurrence of an element:


    a.remove(3)

Reversing the order of the elements:


    a.reverse()

Sorting the elements in ascending order:


    a.sort()

Sorting the elements in descending order:


    a.sort(reverse=True)
