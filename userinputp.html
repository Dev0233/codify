Getting User Input in Python



There are different ways to take user input in Python, depending on the use case. Here are some common methods:



1. Using the input() function



The input() function allows the user to enter data from the keyboard. The function returns a string, which can be converted to the desired data type. Here's an example:

                
            name = input("Enter your name: ")
            age = int(input("Enter your age: "))
            print("Hello, " + name + "! You are " + str(age) + " years old.")

In this example, the user is asked to enter their name and age. The input() function reads the input from the keyboard and stores it in the name and age variables, respectively. The int() function is used to convert the age to an integer, which is required for arithmetic operations. The print() function displays a message that includes the user's name and age.


2. Using command line arguments



Command line arguments can be used to pass data to a Python script. The sys.argv list contains the command line arguments, with the first argument being the name of the script. Here's an example:

                
            import sys

            name = sys.argv[1]
            age = int(sys.argv[2])
            print("Hello, " + name + "! You are " + str(age) + " years old.")

In this example, the user's name and age are passed as command line arguments. The sys.argv list is used to access the arguments, with the first argument being sys.argv[0], which is the name of the script. The second argument (sys.argv[1]) is the user's name, and the third argument (sys.argv[2]) is the user's age. The int() function is used to convert the age to an integer, which is required for arithmetic operations. The print() function displays a message that includes the user's name and age.
