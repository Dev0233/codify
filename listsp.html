Lists in Python



In Python, a list is a mutable, ordered collection of elements that can be of any data type. Lists are created using square brackets [ ], with each element separated by a comma.

Here is an example of a list:

my_list = [1, 'apple', True, 3.14]

In this example, we have a list with four elements: an integer, a string, a boolean, and a float. Lists can also contain other lists as elements, allowing for nested data structures.

Lists are a very versatile data type in Python and have many built-in functions that can be used to manipulate them. Here are some of the most commonly used functions with examples:

  • append() - Adds an element to the end of a list.
    • 		my_list = [1, 2, 3]
		my_list.append(4)
		print(my_list) # Output: [1, 2, 3, 4]
  • insert() - Inserts an element at a specific index in a list.
    • 		my_list = [1, 2, 3]
		my_list.insert(1, 4)
		print(my_list) # Output: [1, 4, 2, 3]
  • remove() - Removes the first occurrence of an element from a list.
    • 		my_list = [1, 2, 3, 4]
		my_list.remove(2)
		print(my_list) # Output: [1, 3, 4]
  • pop() - Removes and returns the element at a specific index in a list.
    • 		my_list = [1, 2, 3, 4]
		element = my_list.pop(2)
		print(my_list) # Output: [1, 2, 4]
		print(element) # Output: 3
  • sort() - Sorts the elements in a list in ascending order.
    • 		my_list = [4, 1, 3, 2]
		my_list.sort()
		print(my_list) # Output: [1, 2, 3, 4]
  • reverse() - Reverses the order of the elements in a list.
    • 		my_list = [1, 2, 3, 4]
		my_list.reverse()
		print(my_list) # Output: [4, 3, 2, 1]
  • count() - Counts the number of occurrences of an element in a list.
    • 		my_list = [1, 2, 2, 3, 3, 3]
		count = my_list.count(2)
		print(count) # Output: 2
  • index() - Returns the index of the first occurrence of an element in a list.
    • 		my_list = [1, 2, 3, 4, 3, 2, 1]
		index = my_list.index(3)
		print(index) # Output: 2
  • len() - Returns the number of elements in a list.
    • 		my_list = [1, 2, 3, 4]
		length = len(my_list)
		print(length) # Output: 4
  • These are just a few of the many functions available for working with lists in Python.
