Arrays in C++

In C++, an array is a collection of elements of the same type, stored in contiguous memory locations. Arrays are useful when you want to store multiple values of the same data type and access them using an index.

C++ Array Examples

Example 1:declaring and accessing elements of an array in C++:

				
					#include <iostream>

					int main() {
						// Declare an array of integers
						int numbers[5];

						// Assign values to the elements
						numbers[0] = 10;
						numbers[1] = 20;
						numbers[2] = 30;
						numbers[3] = 40;
						numbers[4] = 50;

						// Access and print the elements
						for (int i = 0; i < 5; i++) {
							std::cout << "Element at index " <<
							 i << ": " << numbers[i] << std::endl;
						}

						return 0;
					}

The above code declares an integer array named "numbers" with a size of 5. It assigns values to each element of the array and then uses a loop to access and print the elements.

The output of the above code will be: 

				
					Element at index 0: 10
					Element at index 1: 20
					Element at index 2: 30
					Element at index 3: 40
					Element at index 4: 50

Arrays in C++ provide a convenient way to store and manipulate collections of data. They can be used in various scenarios such as storing a list of numbers, characters, or objects.

For more details and advanced operations on arrays in C++, you can refer to the C++ Arrays Documentation.



Example 2: Declaring and Accessing Array Elements

					
					#include 

					int main() {
						// Declare and initialize an array
						int numbers[] = {1, 2, 3, 4, 5};

						// Accessing array elements
						std::cout << "First element: " << numbers[0] << std::endl;
						std::cout << "Second element: " << numbers[1] << std::endl;

						return 0;
					}

Example 3: Modifying Array Elements

					
							#include 
							int main() {
								int numbers[] = {1, 2, 3, 4, 5};

								// Modifying array elements
								numbers[2] = 10;

								// Printing array elements
								std::cout << "Array elements: ";
								for (int i = 0; i < 5; i++) {
									std::cout << numbers[i] << " ";
								}
								std::cout << std::endl;

								return 0;
							}

Example 4: Array Length and Iteration

					
							#include 

							int main() {
								int numbers[] = {1, 2, 3, 4, 5};
								int length = sizeof(numbers) / sizeof(numbers[0]);

								// Iterating through the array
								std::cout << "Array elements: ";
								for (int i = 0; i < length; i++) {
									std::cout << numbers[i] << " ";
								}
								std::cout << std::endl;

								return 0;
							}
