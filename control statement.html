C++ Control Statements

Control statements are an essential part of programming, allowing the programmer to make decisions and execute code based on those decisions. There are three types of control statements in C++:

  • Conditional statements
  • Loop statements
  • Jump statements

Conditional Statements

Conditional statements execute code based on the evaluation of a boolean expression. The most common conditional statements in C++ are:

  • if statement
  • if-else statement
  • switch statement

if Statement

The if statement is used to execute code if a condition is true:


            if (condition) {
                // code to execute if condition is true
            }

Example: if Statement

Here's an example of an if statement that prints "Hello World" if a variable x is equal to 1:


            #include <iostream>
            using namespace std;
            
            int main() 
            {
                int x = 1;
                if (x == 1) {
                cout << "Hello World";
            }
                return 0;
            }

if-else Statement

The if-else statement is used to execute code if a condition is true, and different code if the condition is false:


                if (condition) {
                    // code to execute if condition is true
                } 
                else {
                // code to execute if condition is false
                }

Example: if-else Statement

Here's an example of an if-else statement that prints "Hello World" if a variable x is equal to 1, and "Goodbye World" if x is not equal to 1:


                #include 
                using namespace std;
            
                int main() 
                {
                    int x = 2;
                    if (x == 1) 
                    {
                        cout << "Hello World";
                    }
                    else 
                    {
                        cout << "Goodbye World";
                    }
                return 0;
                }

switch Statement

The switch statement is used to execute different code based on the value of a variable:


            switch (variable) 
            {
                case value1:
                    // code to execute if variable is equal to value1
                    break;
                case value2:
                    // code to execute if variable is equal to value2
                    break;
                default:
                    // code to execute if variable is not equal to any of the values
            }

Example: switch Statement

Here's an example of a switch statement that prints a message based on the value of a var x:


                #include 
                using namespace std;
            
                int main() 
                {
                    int x = 2;
                    switch(x) {
                    case 1:
                        cout << "x is equal to 1";
                        break;
                    case 2:
                        cout << "x is equal to 2";
                        break;
                    default:
                        cout << "x is not equal to 1 or 2";
                        break;
                        }
                    return 0;
                }

Example:

The switch statement in C++ is a type of conditional statement that allows you to choose between several options based on the value of a single variable. Here is an example of a switch statement:


				switch(day) {
					case 1:
						cout << "Monday" << endl;
						break;
					case 2:
						cout << "Tuesday" << endl;
						break;
					case 3:
						cout << "Wednesday" << endl;
						break;
					default:
						cout << "Invalid day" << endl;
						break;
					}

In this example, the variable "day" is being tested against three possible values using the switch statement. If "day" is equal to 1, the code inside the first case block will execute. If "day" is equal to 2, the code inside the second case block will execute. If "day" is equal to 3, the code inside the third case block will execute. If "day" does not match any of the cases, the code inside the default block will execute.

The break statement is used after each case to prevent the code from falling through to the next case. If the break statement is not used, all the code after the matching case will execute until a break statement or the end of the switch statement is reached.
