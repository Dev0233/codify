Sets in Python



In Python, a set is an unordered collection of unique elements. Sets are defined by enclosing a comma-separated sequence of values in curly braces {}.



Creating a Set



Here's an example of how to create a set in Python:

    my_set = {1, 2, 3, 4, 5}

In this example, we define a set my_set that contains the integers 1 through 5.



Set Properties



Sets in Python have several useful properties, including:

  • Uniqueness: Sets contain only unique elements, meaning that there are no duplicates. If we try to add a duplicate element to a set, it will simply be ignored.
  • Order: Sets are unordered, meaning that the elements of a set have no specific order. We cannot access elements of a set by index.
  • Mutable: Sets are mutable, meaning that we can add and remove elements from a set after it has been created.


Working with Sets



Here are some examples of working with sets in Python:

    my_set = {1, 2, 3, 4, 5}
	Adding elements to a set
	my_set.add(6)
	my_set.add(7)
	
	Removing elements from a set
	my_set.remove(4)
	
	Checking if an element is in a set
	print(3 in my_set) # Output: True
	
	Iterating over a set
	for element in my_set:
	print(element)

In this example, we first create a set my_set containing the integers 1 through 5. We then add the integers 6 and 7 to the set using the add() method. We remove the element 4 from the set using the remove() method. We then check if the element 3 is in the set using the in operator. Finally, we iterate over the set using a for loop and print each element.



Set Operations



Python sets also support several set operations, such as union, intersection, and difference. Here are some examples:

    set1 = {1, 2, 3}
		set2 = {2, 3, 4}

		Union
		print(set1 | set2) # Output: {1, 2, 3, 4}
		
		Intersection
		print(set1 & set2) # Output: {2, 3}
		
		Difference
		print(set1 - set2) # Output: {1}

In this example, we define two sets set1 and set2 containing some overlapping elements. We then compute the union of the sets using the | operator, which combines all the unique elements of both sets. We compute the intersection of the sets using the & operator, which returns only the elements that are common to both sets. We compute the difference of the sets using the - operator, which returns only the elements that are in set1 but not in set2.

Set Functions

Python sets also have several useful built-in functions:

  • len(): Returns the number of elements in a set.
  • max(): Returns the largest element in a set (if the elements are numeric).
  • min(): Returns the smallest element in a set (if the elements are numeric).
  • sum(): Returns the sum of the elements in a set (if the elements are numeric).

Here are some examples of using these functions:

    # Creating two sets
          set1 = {1, 2, 3}
          set2 = {3, 4, 5}
          
          # Union of set1 and set2
          print(set1.union(set2)) # Output: {1, 2, 3, 4, 5}
          
          # Intersection of set1 and set2
          print(set1.intersection(set2)) # Output: {3}
          
          # Difference of set1 and set2
          print(set1.difference(set2)) # Output: {1, 2}
          
          set3 = set1 | set2  # union
          set4 = set1 & set2  # intersection
          set5 = set1 - set2  # difference
          set6 = set1 ^ set2  # symmetric difference
          
          print(set3)  # {1, 2, 3, 4}
          print(set4)  # {2, 3}
          print(set5)  # {1}
          print(set6)  # {1, 4}
