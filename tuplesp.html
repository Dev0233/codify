Tuples in Python

A tuple is an ordered collection of elements, similar to a list, but tuples are immutable, which means they cannot be changed once created. Tuples are defined using parentheses, and individual elements are separated by commas.

Creating Tuples

To create a tuple, you can enclose a sequence of values in parentheses:

    		
			# create a tuple
			my_tuple = (1, 2, 3, "hello")
			print(my_tuple)  # (1, 2, 3, 'hello')

Accessing Elements of a Tuple

Elements of a tuple can be accessed using indexing. Indexing is zero-based, which means the first element has an index of 0.

    		
			# accessing elements of a tuple
			my_tuple = (1, 2, 3, "hello")
			print(my_tuple[0])  # 1
			print(my_tuple[3])  # 'hello'

Iterating Over a Tuple

You can use a for loop to iterate over the elements of a tuple:

    		
			# iterating over a tuple
			my_tuple = (1, 2, 3, "hello")
			for element in my_tuple:
				print(element)

    Tuple Methods

    Tuples have a few methods that can be used to perform certain operations:

    • count(): Returns the number of times a specified element appears in the tuple.
    • index(): Returns the index of the first occurrence of a specified element in the tuple.
    		
			# using tuple methods
			my_tuple = (1, 2, 3, "hello")
			print(my_tuple.count(2))  # 1
			print(my_tuple.index("hello"))  # 3

    Tuple Functions

    Python provides several built-in functions for working with tuples:

    len()

    The len() function returns the number of elements in a tuple:

    my_tuple = (1, "apple", True, 3.14)
		print(len(my_tuple))  # Output: 4

    max() and min()

    The max() and min() functions return the largest and smallest elements in a tuple, respectively:

    my_tuple = (1, 5, 3, 2, 4)
		print(max(my_tuple))  # Output: 5
		print(min(my_tuple))  # Output: 1

    sum()

    The sum() function returns the sum of all elements in a tuple:

    my_tuple = (1, 5, 3, 2, 4)
		print(sum(my_tuple))  # Output: 15

    count()

    The count() function returns the number of times a specified element appears in a tuple:

    my_tuple = (1, 5, 3, 2, 4)
		print(my_tuple.count(5))  # Output: 1

    index()

    The index() function returns the index of the first occurrence of a specified element in a tuple:

    my_tuple = (1, 5, 3, 2, 4)
		print(my_tuple.index(3))  # Output: 2
