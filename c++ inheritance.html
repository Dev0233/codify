C++ Inheritance

Inheritance is a fundamental concept in object-oriented programming that allows you to create new classes (derived classes) from existing classes (base classes). The derived classes inherit the properties and behaviors of the base class, and can also add new features or modify the existing ones. In C++, inheritance is implemented using the following syntax:


                class DerivedClass : public BaseClass {
                    // Additional members and functions of the derived class
                };

Example: Inheriting from a Base Class

Here's an example that demonstrates inheritance in C++. We have a base class called Shape, which has a member function printArea(). We then create a derived class called Rectangle that inherits from the Shape class and adds its own member variable length and member function calculateArea():


                    #include <iostream>
                    using namespace std;

                    // Base class
                    class Shape {
                    public:
                    void printArea() {
                    cout << "This is the base class." << endl;
                    }
                    };

                    // Derived class
                    class Rectangle : public Shape {
                    private:
                    int length;
                    int width;

                    public:
                    Rectangle(int l, int w) {
                    length = l;
                    width = w;
                    }
                    void calculateArea() {
                        int area = length * width;
                        cout << "Area: " << area << endl;
                    }
                };

                int main() {
                // Create an object of the derived class
                Rectangle rectangle(5, 7);
                // Access the base class member function
                    rectangle.printArea();

                    // Access the derived class member function
                    rectangle.calculateArea();

                    return 0;
                }

In this example, the Rectangle class inherits from the Shape class using the publicaccess specifier. This means that the public members of the Shape class are accessible in the Rectangle class. The Rectangle class adds its own members length and width and defines the calculateArea() member function to calculate the area of the rectangle. The printArea() member function is accessed from the base class.

When the program is run, it will output:


            This is the base class.
            Area: 35

The program demonstrates how the derived class inherits the printArea() member function from the base class and adds its own functionality with the calculateArea() member function.


Types of Inheritance

C++ supports different types of inheritance:

  • Single Inheritance:

    A derived class inherits from a single base class.

  • Multiple Inheritance:

    A derived class inherits from multiple base classes.

  • Multilevel Inheritance:

    A derived class inherits from another derived class.

  • Hierarchical Inheritance:

    Multiple derived classes inherit from a single base class.

  • Hybrid Inheritance:

    Combination of multiple types of inheritance.

Each type of inheritance has its own use cases and implications, allowing you to create complex class hierarchies and relationships.
