Data Types in Python

Python has several built-in data types, which can be divided into the following categories:


Numeric Data Types:

  • Integers (int): These are whole numbers (positive, negative, or zero) that do not have a decimal point. For example:
    • 
				x = 5
				y = -10
				z = 0
  • Floating-point numbers (float): These are numbers that have a decimal point. For example:
    • 
				a = 2.5
				b = -3.75
				c = 0.0
  • Complex numbers (complex): These are numbers that have a real and imaginary part, expressed as a + bj. For example:
    • 
				d = 2 + 3j
				e = -4j
				f = 1.5 - 2j

Boolean Data Type:

  • Boolean (bool): This data type can only have two values, True or False. For example:
    • 
				x = True
				y = False

Sequence Data Types:

  • Strings (str): These are sequences of characters, enclosed in either single or double quotes. For example:
    • 
				s1 = 'hello'
				s2 = "world"
				s3 = "Python's cool"
				s4 = 'He said, "Hello!"'
  • Lists (list): These are ordered sequences of values, enclosed in square brackets and separated by commas. For example:
    • 
				lst1 = [1, 2, 3]
				lst2 = ['a', 'b', 'c']
				lst3 = [1, 'hello', 2.5, True]
  • Tuples (tuple): These are ordered sequences of values, enclosed in parentheses and separated by commas. Tuples are immutable, meaning their values cannot be changed once they are created. For example:
    • 
				t1 = (1, 2, 3)
				t2 = ('a', 'b', 'c')
				t3 = (1, 'hello', 2.5, True)

Mapping Data Type:

  • Dictionaries (dict): These are unordered collections of key-value pairs, enclosed in curly braces and separated by commas. The keys must be unique and immutable, while the values can be of any data type. For example:
    • 
				d1 = {'name': 'Alice', 'age': 25, 'city': 'New York'}
				d2 = {1: 'apple', 2: 'banana', 3: 'cherry'}
				d3 = {'a': 1, 'b': 2, 'c': 3}

Set Data Type:

  • Sets (set): These are unordered collections of unique elements,enclosed in curly braces and separated by commas. For example: 
    • 
				s1 = {1, 2, 3}
				s2 = {'a', 'b', 'c'}
				s3 = {1, 'hello', 2.5, True}

Sequence Containers:

  • Sequence Containers are collections of ordered sequences.
  • Bytearrays (bytearray): These are mutable sequences of bytes, which can be modified after creation. For example:
    • 
				ba1 = bytearray(b'hello')
				ba1[0] = ord('H')
				print(ba1)
  • Bytes (bytes): These are immutable sequences of bytes, which cannot be modified after creation. For example:
    • 
				b1 = b'hello'
				b2 = bytes([72, 101, 108, 108, 111])

NoneType Data Type:

  • None (NoneType): This data type has only one value, None. It is used to represent the absence of a value or to indicate that a variable does not have a value. For example:
    • x = None
  • These are all the major data types in Python. It is important to understand the different data types and how to use them, as they are the building blocks of any program.
